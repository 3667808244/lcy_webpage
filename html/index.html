类型系统

lcy 语言的类型系统是静态的，支持基本类型、复合类型。

以下是 lcy-lang 中常用的类型注解:

注:T为正常类型,S为大小,E为错误类型

名称 类型注解 示例 适用版本
整数 int

num : int = 1 ;

>=0.0.1
浮点数 float

num : float = 2.0 ;

>=0.0.1
布尔值 bool

flag : bool = true ;

>=0.0.1
字符串 str

text : str = "Hello, World!" ;

>=0.0.1
数组 arr[T,S]

arr : arr[int, 5] = [1, 2, 3, 4, 5] ;

>=0.0.1
元组 tup[T1,T2,T3......]

tup : tup[int, str] = (1, "Hello") ;

>=0.0.1
可空值 opt[T]

value1 : opt[int] = null ;

value2 : opt[int] = some(2) ;

>=0.0.1
返回值（含错误） ret[T,E]

result : ret[int,str] = ok(1) ;

error : ret[int,str] = err("Error message") ;

>=0.0.1
映射 map[K,V]

mapping : map[str,int] = [

  key1 : int = 1 ;

  key2 : int = 2 ;

];

>=0.0.2
指针 ptr[T]

num : int = 10 ;

ptr : ptr[int] = & num ;

ptr_2 : ptr[ptr[int]] = &ptr ;

# 解析时会直接把num的值覆盖掉ptr的值

# 然后在把ptr_2用ptr覆盖

>=0.0.2
对象 obj

person : obj = {

  name : str = "Alice" ;

  age : int = 30 ;

}

>=0.0.1
自动推断 auto

auto_value : auto = "Hello, World!" ;

>=0.0.2

转换标记(change_sign)

change_sign可以让lcy解析器预处理数据

change_sign中只能含有大小写字母,数字,连字符,下划线

change_sign使用前需要把实现的方法(函数)注册到对应的change_sing上

如果没有注册一律以注册到@__default_change_sign__的方法处理

以下是默认实现的change_sing

change_sign 描述 适用类型 适用版本
@__default_change_sign__ 默认转换标记，直接返回原数据 所有类型 >=0.0.1
@trim 去除字符串首尾的空格,换行符,制表符 str >=0.0.1
@to_lower 将字符串转换为小写 str >=0.0.1
@to_upper 将字符串转换为大写 str >=0.0.1
@to_path 将字符串,元组,列表转换为Path对象 str, tup[srt,str.str...], arr[str,S] >=0.0.1
@to_time 将时间戳转换为time对象 int >=0.0.1
@to_url 将字符串转换为URL对象 str >=0.0.1
@insert_other_file 将其他文件(*.ini,*,json,*.lcy,*.toml,*.yml,*.prop,
*.cfg,*.txt(以lcy语法编写),*.xml(会丢失标签属性))
的解析后内容插入到当前文件中 		同@to_path >=0.0.1
@escape 将字符串中的转义符转义 str >=0.0.2

高级特性

类型:auto

类型auto会根据赋值的内容自动推断类型

auto类型可以用于所有lcy-lang所支持的类型

示例

标准语法

name : str = "lcy-lang" @to_upper ;

version : float = 1.0 ;

features : arr[str, 3] = ["type system", "syntax", "example"] ;

config : obj = {

  enabled : bool = true ;

  timeout : int = 30 ;

};

解析为:(py)

{

    'name' : 'LCY-LANG' ,

    'version' : 1.0 ,

    'features' : ['type system', 'syntax', 'example'] ,

    'config' : {

        'enabled' : True ,

        'timeout' : 30 ,

    }

}

指针示例

num : int = 10 ;

ptr : ptr[int] = &num ;

ptr2 : ptr[ptr[int]] = &ptr ;

解析为:(py)

{

    'num' : 10

    'ptr' : 10

    'ptr2' : 10

}
